# 🏢 PracticasEmpresa1

Mini aplicación de ejemplo con **Spring Boot 3**, **H2 Database** y **REST API** para gestionar usuarios (login y registro).

---

## 🚀 Tecnologías usadas

- **Java 17**
- **Spring Boot 3.5.6**
- **Spring Data JPA**
- **H2 Database (en memoria)**
- **Maven**
- **REST API (JSON)**
